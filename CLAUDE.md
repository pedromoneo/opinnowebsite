# Opinno Website - Project Instructions

## Deployment
- **ALWAYS deploy using Firebase**, never Vercel
- Deploy command: `firebase deploy` (from project root)
- Firebase hosting region: europe-west1
- Firestore database: (default), region: europe-west1

## Tech Stack
- Next.js with Firebase Hosting (web frameworks integration)
- Firestore for database
- Firebase Auth (email/password)
